## Templates
- Name blocks with lowercase and underscores.
```
{% block lowercase_with_underscores %}
```

- In endblocks, add the name of the block they close.
```
{% block foo_bar %}
	...
{% endblock foo_bar %}
```

- Indent everything within template tags.
```
{% block foo_bar %}
	<html-tag></html-tag>
	{% if foo %}
		<html-tag></html-tag>
	{% else %}
		<html-tag></html-tag>
	{% endif %}	...
{% endblock foo_bar %}
```
